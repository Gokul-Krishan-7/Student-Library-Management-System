# Student-Library-Management-System

To preview and run the project on your device:
  1) Open project folder in <a href="https://spring.io/tools">spring tool suite</a>
  2) Download file and ready create folder ,then export folder to spring tool suite application
  3) From the File menu, select Import,The Select screen opens.
  4) Expand Maven and select Existing Maven Projects.
  5) Click Run `play` to view project in browser
